About ArcWeather

This project was developed as a submission for the Hack With India contest organized by the FEDF department at KLH Bachupally. It aims to provide an interactive platform where users can retrieve real-time weather and time data for any location on the globe.

Technologies used in this project include Leaflet.js for the interactive world map, and the OpenWeather API for fetching live weather data. The interface is enhanced with a responsive dark/light mode toggle, modern UI styling, and real-time clock visualization (both digital and analog).

Additionally, this project taught us how to effectively work with JSON structures, manage asynchronous API calls, and integrate Python Flask with JavaScript for a seamless full-stack application experience. We also learned the importance of user experience, frontend/backend communication, and designing modular, scalable codebases.
